---
title: "mobile:install"
description: "Install NativePHP for Mobile"
---

<Warning>
This command requires that you have a license from NativePHP.
If you do not have a license, you can purchase one from [NativePHP for Mobile](https://nativephp.com/mobile).
</Warning>

This command will bootstrap your existing Laravel app with the NativePHP for Mobile package, ready for you to build your mobile app.

If you have not previously installed the NativePHP for Mobile package, you will be prompted to input your license details.

## Command

Syntax:

```bash
nativecli mobile:install
```

## Options

There are no options for this command.
